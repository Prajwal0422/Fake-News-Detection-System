# Fake News Detection System

## 📌 Project Overview
This project detects whether a news article is Fake or Real using Natural Language Processing (NLP) and Machine Learning.

The system preprocesses news articles, converts text into numerical vectors using TF-IDF, and classifies the news using Logistic Regression.

---

## 📌 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP
- TF-IDF
- Matplotlib
- Seaborn

---

## 📌 Machine Learning Algorithm
- Logistic Regression

Fake and Real News Dataset
